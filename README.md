# 카서유기 - 실시간 통신 퀴즈 게임
카서유기는 4글자, 영화, 인물, 랜덤 퀴즈를 모두 즐길 수 있는 카이스트 신서유기 게임입니다.

# 팀원
- 이중훈(KAIST)
- 이경민(KAIST)
- 최지민(성균관대학교)

# 개발 환경
- Frontend: React Native, Typescript
- Backend: Node.js express (with Socket.io)
- Cloud: AWS
- DB: AWS RDS MySQL
